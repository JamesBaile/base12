# node-boilerplate
      
  Structure for your [node.js](http://nodejs.org) project.

## Installation

    git clone -b edge https://github.com/skookum/node-boilerplate.git .
    cd node-boilerplate
    sudo npm install
    sudo node server.js

## Adding libraries

  Add to 'dependencies' in /package.json, then:
  
    sudo npm install
    
## Included modules:

  - Express
  - Jade
  - Stylus
  - Connect-Timeout
  - Cluster
  - Express-Messages
